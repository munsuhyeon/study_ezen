변수 선언 방식
원하는 계산을 하려면 원하는 데이터를 컴퓨터 메모리에 할당해햐한다
변수의 선언 방식에 따라 데이터를 입력해야 하는데
자바스크립트 변수 선언 방식은 3가지가 있다 var let const

var a = 10; 이런식으로 변수이름 a 앞에 var let const 중 하나를 사용하면 된다
var 방식은 변수의 재선언이 가능한데
var a=8;
var 1=10; 이런식으로 밑에서 재설정하기
a라는 이름의 변수가 특정 계산을 하다가 몇백줄 밑 코드에 다른 사람이 같은 이름으로 전혀 다른 연산을 같은 이름의 변수를 써서 진행한다고 가정해보자
그러면 코드는 완전히 어그러질 것이다
그래서 모던자바스크립트(ES6)이상부터 변수의 재선언을 막는 let과 const 변수 선언 방식을 추가하였다

변수의 이름 a,b같은 거를 식별자라고 한다. 컴퓨터의 메모리는 추상적이므로 사람이 이해할 수 있도록 데이터,함수,
혹은 객체에 이름을 부여한 것을 식별자라고 한다
연산자: 연산 작업을 위해 사용하는 기호 + * / -
표현식: 10+20 같은 수식이 표현되는걸 표현식이라 한다
자바스크립트에서 코드 줄이 끝나면 세미콜론을 붙인다.
var a = 8; 이런식으로 코드의 줄이 끝나면 붙여준다

============================================================
1. 자바스크립트는 홈페이지 만들 때 왜 필요한가?
html, css는 원칙적으로 정적 웹사이트만 구현 가능하다
한번 위 코드들이 브라우저에서 해석되어 화면에 표시되면 수정할 수 없다
이를 동적 웹처럼 작동하게 하려면 자바스크립트 언어가 필요하다

2. 자바스크립트는 어떨 때 쓰는가?
자바스크립트는 풀스택 언어로 진화했다. 홈페이지 만드는 언어 뿐 아니라
서버를 구현하는 백엔드 언어로도 사용할 수 있다(node.js)
화면을 구현하는 프론트 엔드 파트, 서버와 데이터베이스를 처리하는 백엔드 모두 자바스크립트로 구현 가능하다

3. 자바스크립트 사용법
크롬과 같은 웹 브라우저를 쓰던디 node.js 실행기를 쓰던지 해야한다. 그냥 코드만 실행하지 못한다
<브라우저 통해서 자바스크립트 실행법 3가지>
1. 크롬 브라우저가서 f12 개발자도구 실행 console탭가서 실행하는방법
2. script태그를 만들어서 자바스크립트 파일 링크를 제공하는 법
3. script태그 안에 자바스크립트 코드를 직접 입력하는 방법 

4. 자바스크립트 사용 시 주의사항
어디서 에러가 났는지 프로그램 시작전에 에러를 표시하지 않고 브라우저가 무리해서 코드를 실행하려 한다
그래서 코드 어디에서 정확히 에러가 났는지 찾기 까다롭다

5.변수 선언
var방식은 변수의 재선언 가능한 현상으로 인해 다른 기능의 함수가 같은 이름의 변수로 재선언되는 사고가 날 가능성이 존재한다
변수의 재 선언을 막는 것이 let과 const방식이다
let: 변수의 재선언이 불가능하지만 재할당은 가능하다
let a=8;
a=11;
변수 재선언이 아닌 범위 내에서 데이터 교체는 가능하다
const: 재선언,재할당이 불가능하다. 
단 배열 데이터안의 세부 데이터 변경은 가능하다
const bbb=[1,2,4];
bbb[1]=1; (이런식으로 인덱스로 데이터를 집어서 변경해야한다)
bbb=[1,1,4]

<변수의 재선언>
let num=10;
let num=8;
<변수의 재할당>
let num=10;
num=8;

데이터는 데이터 타입이 있는데 크게 숫자형과 문자열이 있다
자바스크립트와 파이썬은 따옴표 사용에 유동적이다
유니코드 변환여부에 따라 큰따옴표 작은따옴표 구분됨
